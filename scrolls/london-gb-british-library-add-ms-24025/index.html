London, GB, British Library, add MS 24025

Repository: British Library, London GB

Date: 1200 - 1300

Dimensions: 315cm x 27.9cm

Type: Trees of Christ

Language(s): Latin

Shelfmark: add MS 24025

Provenance: Unknown Provenance

Contents: Peter of Poitiers Compendium historiae in genealogia Christi. Begins: "Considerans historie prolixitatem."

Description: Vellum roll with colored drawings and medallion portraits.
